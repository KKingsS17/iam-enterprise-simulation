# Access Governance Module

## Overview

This module simulates an enterprise Identity and Access Management (IAM) governance framework using Microsoft Entra ID.

It includes:

- Role-Based Access Control (RBAC)
- Access Reviews
- Audit simulation
- Automated access reporting
- Dashboard visualization

## Components

### Documentation

- Business rules and RBAC model
- Risk classification
- Audit simulation scenarios

### Automation

- PowerShell script to generate access reports

### Reports

- IAM access audit report (CSV)
- Executive summary
- Power BI dashboard

### Evidence

- Access review results
- Screenshots of remediation actions

## Key Capabilities
- Detection of privileged access
- Identification of inactive accounts
- Risk-based access classification
- Role inheritance simulation
- Audit-ready reporting

## Outcome

This module demonstrates the implementation of IAM governance controls aligned with enterprise security and audit practices.
